## Who I am

I'm a student and freelancer at the intersection of security, development, and infrastructure. I build offensive tools to understand how systems break — then use that knowledge to help build things that don't. I work across the full stack: Go CLIs, Python scripts, Docker environments, hardened Linux boxes.

If you're looking for someone who thinks like an attacker and builds like a developer — that's me.
